## Project Description
### This project focuses on cleaning, merging, and analyzing over a decade of financial crime data to uncover actionable insights. The goal is to support data-driven decision-making that could help reduce crime rates through targeted policy interventions.

[dashboard.pdf](https://github.com/user-attachments/files/23564112/dashboard.pdf)



## Aim 
### To develop a clear, insightful, and visually engaging dashboard that effectively communicates trends and patterns in financial crime across regions and time.

## Objectives
### - Clean and merge multiple datasets into a unified structure
### - Build an interactive dashboard using Excel
### - Communicate key insights through visual storytelling

## Process Overview

### 1. Data Cleaning
### Tool Used: Excel Power Query

### - Removed duplicates and handled missing values

### - Ensured consistency across all sheets

### - Merged data from 2012 to 2024 into a structured dataset

### 2. Data Analysis
### - Identified correlations and trends across years and regions

### - Focused on crime types and law enforcement distribution

### 3. Dashboard Creation
### - Used pivot tables and slicers for interactivity

### - Visualized data using bar charts, line graphs, and pie charts

### - Enabled dynamic filtering by year, crime type, and police force

### 4. Insight Summarization
### - Provided narrative explanations for observed patterns

### - Highlighted anomalies and potential causes

## Key Insights
### - Crime Rate Trends: High crime rate in 2012 followed by a period of stability; sharp drop observed in 2024.

### - Metropolitan Police: Handled the highest number of crimes, especially theft offences â€” nearly triple that of other forces.

### - Crime Distribution: Theft offences were the most common, followed by violence against persons and fraud offences.

### - Fraud Hotspots: Action Fraud & CIFAS Forces dealt with the majority of fraud cases, indicating regional concentration.

## Recommendations
### - Policy Implementation: Introduce stricter penalties for theft and fraud offences.

### - Resource Allocation: Increase support for forces handling high volumes of specific crimes.

### - Further Analysis: Investigate the cause of the drop in 2024.
